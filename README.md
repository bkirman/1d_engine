# 1D Micropython Game Engine for EMF Workshop on Making Games with Fairy Lights

Written and maintained by [Ben Kirman](https://ben.kirman.org).

# How to use:
These files are designed for the [Pimoroni Plasma 2350W](https://shop.pimoroni.com/products/plasma-2350-w) board. Replace the files that are preinstalled on the board with the contents of this repository.

You can always re-flash the board back to stock later, or see Pimoroni's [repository of example code on GitHub](https://github.com/pimoroni/plasma/tree/main/examples).
You also probably want to solder header pins to TX and GND (nearest USB port) for connecting the controller button more easily.

Once you have done this, you are ready to [follow the workshop guide](https://ben.kirman.org/1d) from the start. It includes a shopping list of materials for the workshop.

You are also just welcome to mess around and see what's going on. A note that the workshop is aimed at younger audiences, so I've prioritised readability ahead of maintainability or efficiency. That stuff is overrated anyway :)
